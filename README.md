# agents_course
An introduction to AI Agents - slides and code

## Content of the course

1. Introduction to AI Agents
2. A short history of AI
3. Tool use, memory, context and retrieval
4. Agentic Patterns & architectures
5. Evaluating your agents
6. Agents in production

Possible other subjects:
- Grounding your agents
- Ontology Augmented Generation
- OpenCode, Hermes Agent, Claude Code
- Security, permissions etc.
  
## How this repo is built
- all content is stored in markdown (.md) format
- this repo uses quarto / marp to produce powerpoint slides based on the content of the .md files.
- Claude Code and other AI-tools are used to create content
